X-links Extensions

Add custom features to X-links, including linkification, requests, settings, and detail previews.

X-links provides an API for developers to extend its normal capabilities. As X-links takes care of a lot of compatibility quirks with different scripts, this makes it easy to add features without having to fork or copy code from it.

Extensions are installed as separate userscripts which can then communicate with the main X-links script by using the Extensions API JavaScript file provided.

Available Extensions

  • Nyaa Torrents - Linkify and format nyaa.se links
  • Multimedia - Linkify and format various multimedia links, including:
    • YouTube*
    • Vimeo
    • SoundCloud
    • DailyMotion
    • LiveLeak
    • Vine
    • Twitter
    • Livestream
    • Niconico
    • Github Gist

*Note that the Multimedia extension uses the YouTube API v3 which requires a application key. By default, one is provided, but using it will perform some basic anonymous stat tracking.

The extension tries to keep these requests as clean as possible by removing referrer and cookie information that might be sent, but if you aren't satisfied by this, you can generate your own key for private usage and use it in the settings.
